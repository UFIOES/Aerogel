Aerogel

By: UFIOES

To install, extract Aerogel into \Steam\SteamApps\common\TUG\Mods\
Then add "Mods/Aerogel" to \Steam\SteamApps\common\TUG\Config\mods.txt
e.g.

Mods
{
	"Mods/Aerogel"
	"Game/Dev"
	"Game/Survival"
	"Game/Creative"
	"Game/Core"
}
